# Cloud AI Workspace

Plataforma personal desplegada sobre Kubernetes para acceder remotamente a un entorno de desarrollo con GPU.

## Objetivos iniciales

- Ejecutar Kubernetes mediante k3s sobre WSL2.
- Compartir la GPU NVIDIA RTX 2070 con contenedores.
- Desplegar un entorno Jupyter persistente.
- Acceder desde un portátil mediante una conexión privada segura.
- Automatizar y documentar el despliegue.
